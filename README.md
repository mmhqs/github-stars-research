## github-stars-research 
The open-source software (OSS) movement has transformed the technology industry, enabling global collaboration on an unprecedented scale. Platforms such as GitHub have become central repositories of knowledge and innovation, where popularity metrics, such as the number of ‘stars’, serve as indicators of relevance and adoption. 

However, popularity as an isolated metric does not explain the health or dynamics of a system. This report presents a quantitative analysis of the 1,000 most popular repositories on GitHub in March 2026, investigating whether the fame of these projects correlates with maturity, sustained activity and community engagement.
